# موقع المستشفى (نسخة GitHub Pages)

هذه نسخة ثابتة (Static) من موقع المستشفى، تم تجهيزها لتعمل مباشرة على **GitHub Pages** دون الحاجة إلى خادم (Backend).

## طريقة التشغيل على GitHub:

1. قم بإنشاء مستودع جديد (New Repository) على GitHub.
2. ارفع محتويات هذا المجلد إلى المستودع.
3. اذهب إلى إعدادات المستودع (**Settings**) -> **Pages**.
4. اختر الفرع الرئيسي (**main**) والمجلد الجذر (**/**) ثم اضغط **Save**.

سيكون موقعك متاحاً خلال دقائق على رابط GitHub Pages الخاص بك.
